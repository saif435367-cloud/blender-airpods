This project was created in Blender as a practice in hard-surface modeling. The goal was to improve my ability to model clean, real-world objects with accurate proportions.
I modeled an AirPods set and charging case using a tutorial on youtube by poligon runway . The focus of this project was purely on modeling, not materials, lighting, or texturing.

 What I Learned:
Basic hard-surface modeling techniques
Maintaining clean and simple topology
Matching real-world object proportions
Improving modeling speed and accuracy in Blender

Future Improvement:
Add materials and realistic textures
Improve detail and small surface features
Create a full product render setup
